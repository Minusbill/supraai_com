# 超灵智能（SpuraAI）官网项目

## 项目概述
超灵智能官网，纯静态站点（HTML + CSS + JS），无框架依赖。

## 项目结构
- `index.html` — 首页
- `pages/` — 子页面：services、training、industries、cases、mcp-dev、about、contact
- `css/` — 样式：variables.css（变量）、style.css（主样式）、pages.css（页面样式）
- `js/` — 脚本：components.js（公共组件）、main.js（主逻辑）
- `images/` — 图片资源

## 设计风格
- 深蓝主题
- 文案要求：说人话，去AI味，简洁不冗余
- 卡片描述文字 hover 才显示，页面保持干净
- 导航下拉点击触发

## 业务线
- AI 咨询
- AI 培训（企业AI转型方向，OpenClaw工具支持）
- 定制开发（MCP/Skill 开发）

## 文案原则
- 精简，不堆砌
- 不用"赋能""颠覆""革命性"等空洞词汇
- 用具体场景和效果说话

## 提交规范
- 中文 commit message
- 格式：`类型: 描述`（feat/fix/refactor/revert）
