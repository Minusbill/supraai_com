# supraai_com

超灵智能官网 — 珠三角中小企业 AI 转型一站式服务商

🌐 **域名**: [chaolng.ai](https://chaolng.ai)

## 技术栈

- 纯 HTML / CSS / JS（零依赖）
- CSS 自定义变量（Design Tokens）
- 深蓝色主题 + 暗色模式自动适配
- 响应式设计（Mobile First）
- Intersection Observer 滚动动画

## 目录结构

```
├── index.html          # 主页
├── css/
│   ├── variables.css   # 设计令牌
│   └── style.css       # 主样式
├── js/
│   └── main.js         # 交互逻辑
├── images/
│   └── favicon.svg     # 站点图标
├── sitemap.xml         # 站点地图
└── robots.txt          # 爬虫配置
```

## 页面板块

- 🏠 Hero — 品牌口号 + 数据亮点
- 💡 服务介绍 — AI 咨询 / 项目实施 / Token 代理
- 🏢 企业方案 — RAG 知识库 / Agent 开发 / AI 工作流
- 📂 案例展示 — 真实项目案例与数据
- 🔧 技术服务 — MCP Server & AI Skill 定制开发
- 📚 个人培训 — 课程产品
- 👥 关于我们 — 团队介绍
- 📞 联系我们 — 表单 + 联系方式

## 本地开发

直接用浏览器打开 `index.html` 或使用任意静态服务器：

```bash
npx serve .
# 或
python3 -m http.server 8000
```
